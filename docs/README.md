# Janam Documentation

This folder contains application documentation for engineering, operations, and compliance.

## Documents

- [Architecture](architecture.md): system components, data flow, and integration pattern
- [API Reference](api-reference.md): authentication, HTTP endpoints, and WebSocket channels
- [Deployment Guide](deployment.md): production deployment and rollout checklist
- [Operations Runbook](operations-runbook.md): monitoring, incident response, and troubleshooting
- [Code Compliance](code-compliance.md): coding standards, quality gates, and security/privacy controls

## Quick Start

1. Read [Deployment Guide](deployment.md) to set up production infrastructure.
2. Configure environment variables using `.env.example` as baseline.
3. Follow [API Reference](api-reference.md) to connect clients and connectors.
4. Enforce quality controls from [Code Compliance](code-compliance.md) before release.
